Hey, I’m Chris.

I’m a software engineer who enjoys building random things. whether it’s a web app, a hardware project, or just an idea that needs turning into reality. I like writing code that works (eventually(sometimes)), solving problems that probably didn’t need to be solved, and learning as I go.

My background mixes software engineering, IT support, field technician, and some tinkering on my own time. I’ve worked in a wide range from the hardware installation side to modern JavaScript stacks, and I enjoy experimenting with embedded systems, game development, and anything I can plug into a Raspberry Pi.

Outside of tech, I’m into manga (I'm a strawhat), freestyle skateboarding, indoor climbing, and cooking. My cat Java tends to supervise most of what I do, especially when I’m working on projects late at night.
